Arquitectura

¿Qué es una arquitectura?

En términos generales, la arquitectura es la estructura o diseño de algo, ya sea un edificio, una organización o un sistema. Es el arte y la ciencia de planificar, diseñar y construir, asegurando que los elementos individuales se integren de manera coherente para cumplir con un propósito específico.

Arquitectura de Software

¿Qué es?

La arquitectura de software se refiere a la estructura y diseño de un sistema de software. Es una representación de alto nivel que define cómo los componentes del software interactúan entre sí, cómo se organizan y cómo cumplen con los requisitos funcionales y no funcionales del sistema.

Patrón Cliente-Servidor

Este patrón es uno de los más usados en la arquitectura de software, con gran alcance en el diseño de webs y servicios online.

Patrón de Capas

Este tipo de arquitectura se basa en la subdivisión de la estructura del sistema en diferentes capas, cada una correspondiente a un nivel de abstracción diferente.

Patrón Master-Slave

Este patrón se divide en dos partes: master y slaves. Los slaves realizan las tareas asignadas por el master, quien procesa y almacena los resultados.

Patrón Modelo-Vista-Controlador

El patrón modelo-vista-controlador divide la aplicación en tres partes interactivas: modelo, vista y controlador.

Patrón Broker

Esta arquitectura se usa para estructurar sistemas con componentes desacoplados.
